HTML/CSS 入门指南

2026-05-07 阅读约 10 分钟

如果你是从零开始学习网页开发，这篇文章是为你准备的。我们将从最基础的概念开始，一步步了解 HTML 与 CSS，最终创建一个简洁的导航栏示例。

你只需要两样东西：文本编辑器（推荐 VS Code）和浏览器。不需要安装任何环境。

HTML 是什么

HTML（HyperText Markup Language，超文本标记语言）是网页的骨架。它用"标签"来描述页面的结构和内容。

一个最简单的 HTML 文件长这样：

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>我的第一个网页</title>
</head>
<body>
    <h1>你好，世界！</h1>
    <p>这是我的第一个网页。</p>
</body>
</html>

各部分解释：

  • <!DOCTYPE html>：告诉浏览器这是一个 HTML5 文档
  • <html lang="zh-CN">：根元素，lang 属性指定语言（有助于屏幕阅读器和 SEO）
  • <head>：头部区域，包含元数据，不显示在页面上
  • <body>：主体区域，所有可见内容放在这里
小技巧：在 VS Code 中新建 .html 文件，输入 ! 然后按 Tab 键，可以自动生成完整的 HTML 模板。

常用 HTML 标签

文字与排版

<h1>一级标题（最大）</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<p>这是一段普通段落文字。</p>
<strong>加粗文字</strong>
<em>斜体文字</em>

链接与图片

<!-- 超链接，target="_blank" 在新标签页打开 -->
<a href="https://example.com" target="_blank">点击跳转</a>

<!-- 图片，alt 属性描述图片内容（提升可访问性） -->
<img src="photo.jpg" alt="一张风景照片" width="400">

列表

<!-- 无序列表（圆点） -->
<ul>
    <li>苹果</li>
    <li>香蕉</li>
    <li>橙子</li>
</ul>

<!-- 有序列表（数字） -->
<ol>
    <li>第一步：准备食材</li>
    <li>第二步：开始烹饪</li>
    <li>第三步：装盘享用</li>
</ol>

结构标签

<header>页头区域</header>
<nav>导航菜单区域</nav>
<main>页面主内容区域</main>
<section>内容分区</section>
<article>独立文章内容</article>
<footer>页脚区域</footer>
<div>通用块级容器（占整行）</div>
<span>通用行内容器（不换行）</span>

CSS 是什么

CSS（Cascading Style Sheets，层叠样式表）是网页的外观。它控制颜色、字体、布局、间距等所有视觉效果。

在 HTML 中引入 CSS 有三种方式：

<!-- 方式 1：外部 CSS 文件（推荐，利于维护） -->
<link rel="stylesheet" href="style.css">

<!-- 方式 2：内部样式（写在 <head> 中） -->
<style>
    body { background: #f0f0f0; }
</style>

<!-- 方式 3：行内样式（不推荐，难以维护） -->
<p style="color: red; font-size: 18px;">红色文字</p>

CSS 选择器

/* 标签选择器：匹配所有 <p> 元素 */
p {
    color: #333;
    line-height: 1.6;
}

/* 类选择器：匹配 class="highlight" 的元素 */
.highlight {
    background: yellow;
    font-weight: bold;
}

/* ID 选择器：匹配 id="header" 的元素（页面唯一） */
#header {
    font-size: 24px;
    padding: 20px;
}

/* 后代选择器：nav 内部的 a 元素 */
nav a {
    text-decoration: none;
    color: white;
}

/* 伪类：鼠标悬停状态 */
a:hover {
    color: orange;
    text-decoration: underline;
}

CSS 盒模型

每个 HTML 元素都是一个"盒子"，由四层结构组成：

┌────────────────────────────┐
│          margin            │  ← 外边距（与相邻元素的距离）
│   ┌──────────────────┐     │
│   │     border       │     │  ← 边框
│   │  ┌────────────┐  │     │
│   │  │  padding   │  │     │  ← 内边距（内容与边框的距离）
│   │  │ ┌────────┐ │  │     │
│   │  │ │content │ │  │     │  ← 实际内容
│   │  │ └────────┘ │  │     │
│   │  └────────────┘  │     │
│   └──────────────────┘     │
└────────────────────────────┘
.card {
    width: 300px;           /* 内容宽度 */
    padding: 20px;          /* 内边距（四边相同） */
    border: 2px solid #333; /* 边框 */
    margin: 16px auto;      /* 外边距（auto 实现水平居中） */
    box-sizing: border-box; /* 让 width 包含 padding 和 border */
}
重要：始终在 CSS 开头加上 * { box-sizing: border-box; }，让所有元素使用更直观的尺寸计算方式。这是现代 CSS 的标准做法。

Flexbox 弹性布局

Flexbox 是现代 CSS 中最常用的布局方式，用于在一个方向（水平或垂直）上排列和对齐元素。

.container {
    display: flex;                  /* 启用 Flexbox */
    justify-content: space-between; /* 主轴（水平）对齐 */
    align-items: center;            /* 交叉轴（垂直）对齐 */
    gap: 16px;                      /* 子元素间距 */
    flex-wrap: wrap;                /* 允许换行 */
}

justify-content 常用值：

  • flex-start：靠左对齐（默认）
  • flex-end：靠右对齐
  • center：居中对齐
  • space-between：两端对齐，间距均等
  • space-around：每个元素两侧间距相等

实战：创建导航栏

结合以上知识，创建一个完整的响应式导航栏：

HTML 结构：

<nav class="navbar">
    <div class="logo">MyBlog</div>
    <ul class="nav-links">
        <li><a href="#">首页</a></li>
        <li><a href="#">文章</a></li>
        <li><a href="#">关于</a></li>
    </ul>
</nav>

CSS 样式：

.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 16px 32px;
    background: #1a1a2e;
}

.logo {
    font-size: 20px;
    font-weight: bold;
    color: #e07b58;
}

.nav-links {
    list-style: none;    /* 去掉圆点 */
    display: flex;
    gap: 24px;
    margin: 0;
    padding: 0;
}

.nav-links a {
    color: rgba(255, 255, 255, 0.7);
    text-decoration: none;
    transition: color 0.2s;  /* 平滑颜色过渡 */
}

.nav-links a:hover {
    color: white;
}

下一步学习方向

掌握了 HTML 和 CSS 基础后，可以继续深入：

  • CSS Grid：二维布局系统，适合复杂页面结构
  • 响应式设计：@media 查询，让页面在手机上同样好用
  • CSS 变量：用 --变量名 统一管理颜色和尺寸
  • JavaScript：让页面动起来，实现点击、表单、动画等交互
推荐工具：VS Code + Live Server 插件。保存文件后浏览器自动刷新，实时看到效果，学习效率大幅提升。
